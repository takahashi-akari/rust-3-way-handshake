# Rust-3-Way-Handshake
A simple implementation of the 3-way handshake in Rust.

## Usage
```
$ git clone https://github.com/takahashi-akari/rust-3-way-handshake.git
$ cd rust-3-way-handshake
$ cargo build
$ cargo run
```
